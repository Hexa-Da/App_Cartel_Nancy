# Cartel Nancy

## Description
Cartel Nancy est une application mobile développée à partir du projet CumMap, visant à créer une plateforme de partage et de découverte de lieux d'intérêt dans la ville de Nancy et ses environs.

## Objectifs
- Permettre aux utilisateurs de partager des lieux d'intérêt (restaurants, bars, événements, etc.)
- Offrir une interface intuitive pour la découverte de nouveaux endroits
- Faciliter la navigation et l'exploration de la ville
- Créer une communauté active autour des lieux nancéiens

## Fonctionnalités
- Carte interactive des lieux d'intérêt
- Système d'authentification pour les utilisateurs
- Partage de lieux avec photos et descriptions
- Filtres de recherche par catégories
- Interface administrateur pour la modération

## Technologies utilisées
- React
- TypeScript
- Capacitor pour le développement mobile
- Firebase pour l'authentification et la base de données
- Leaflet pour la cartographie

## Installation
1. Cloner le repository
```bash
git clone https://github.com/Hexa-Da/App_Cartel_Nancy.git
```

2. Installer les dépendances
```bash
npm install
```

3. Lancer l'application en mode développement
```bash
npm run dev
```

4. Pour déployer sur Android
```bash
npm run deploy-android
```
